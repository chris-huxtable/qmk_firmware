# chuxtable's v1.0

Simplified and reduced keymaps.

Install QMK:
```shell
zsh util/qmk_install.sh
```
or:
```shell
brew install qmk/qmk/qmk
```


Compile:
```
make preonic:chuxtable
```

Flash:
```
make preonic:chuxtable:flash
```
